DIAGRAMA ENTIDAD-RELACIÓN (DER)

Modelo de Datos Físico - Base de Datos lachina2_bd (Sistema Centro de Apuestas)

📋 usuario_web
Tabla Principal
Clave Campo Tipo de Dato Detalle / Restricción
PK id_usuario INT NOT NULL, AUTO_INC
INDEX usuario VARCHAR(50) Unique Username
- clave VARCHAR(20) Password de Ingreso
- nivel INT 1=User, 2=Prog, 3=Admin
- estatus INT 1=Activo, 3=Bloqueado
- nombreyapellido VARCHAR(50) Nombre Completo
OPT correo VARCHAR(50) NULL (Opcional)
- ultimo_acceso DATETIME Timestamp Conexión
- ip VARCHAR(20) Dirección IP
- ip_fecha / ip_hora DATE / TIME Fecha/Hora Acceso
📝 suscripcion
Clientes / Perfil
Clave Campo Tipo de Dato Detalle / Restricción
PK id_suscripcion INT NOT NULL, AUTO_INC
FK usuario VARCHAR(100) Relación a usuario_web
OPT correo VARCHAR(100) NULL (Opcional)
- nombre VARCHAR(100) Nombre Registrado
- telefono / celular VARCHAR(50) Contactos Cliente
- estatus INT 0=Reg, 1=Act, 3=Bloq
- fechahora TIMESTAMP Fecha Registro BD
🛡️ historial_conexiones
Auditoría / Logs
Clave Campo Tipo de Dato Detalle / Restricción
PK id INT NOT NULL, AUTO_INC
FK usuario VARCHAR(100) Relación a usuario_web
- nombreyapellido VARCHAR(100) Nombre al conectar
- nivel INT Nivel de Acceso (1..3)
- ip VARCHAR(45) IP Origen Conexión
- fecha_hora TIMESTAMP Fecha y Hora Exacta
- estado VARCHAR(50) Exitoso / Fallido
👤 clientes
Maestro Clientes
Clave Campo Tipo de Dato Detalle / Restricción
PK id_cliente INT NOT NULL, AUTO_INC
- cliente VARCHAR(50) Nombre o Razón Social
- rif VARCHAR(20) Doc. Identidad / RIF
- saldo_actual DECIMAL(14,2) Saldo de Cuenta
🧾 nota_venta
Cabecera Venta
Clave Campo Tipo de Dato Detalle / Restricción
PK id_nota_venta INT NOT NULL, AUTO_INC
FK id_cliente INT Relación a clientes
FK id_vendedor INT Relación a vendedores
- correlativo VARCHAR(20) Número de Factura
- total_neto DECIMAL(14,2) Monto Total Venta
📦 inventario
Catálogo Productos
Clave Campo Tipo de Dato Detalle / Restricción
PK id_inventario INT NOT NULL, AUTO_INC
- codigo VARCHAR(20) Código Producto
- item VARCHAR(100) Descripción Producto
- precio DECIMAL(14,2) Precio de Venta
- existencia DECIMAL(14,2) Stock Disponible
